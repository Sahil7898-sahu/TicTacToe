 Tic Tac Toe Game

A simple and interactive **Tic Tac Toe** game built using HTML, CSS, and JavaScript. This project demonstrates basic game logic, DOM manipulation, and user interaction in a clean and responsive interface.

Features

 Two-player mode (Player X vs Player O)
 Real-time game updates
 Winner detection logic
 Draw condition handling
 Restart game functionality
 Responsive design (works on mobile & desktop)


 Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Tools:** VS Code

 How It Works

1. Player **X** starts the game
2. Players take turns clicking on empty cells
3. The game checks for a winner after every move
4. A player wins if they get **3 in a row** (horizontal, vertical, or diagonal)
5. If all cells are filled and no winner → **Draw**
